# Prompt to give Hermes

Use this when you want another Hermes instance to install and wire up `hermes-tasklane`.

```text
Clone https://github.com/Smokefarmer/hermes-tasklane and install it for this machine.

Requirements:
- install the package
- run the installer or equivalent manual setup
- initialize config and folders
- explain where the inbox folder is
- if systemd user timers are available, install and enable them
- verify with doctor/status
- do not invent repo paths; ask me only if repo paths are actually needed for a real task file
```
